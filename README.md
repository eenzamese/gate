## Installation

Clone the repo
   ```sh
   git clone https://github.com/eenzamese/gate.git
   ```

## Usage

Edit inputs in speed_switch.py
```py
# inputs
APP_TMT = 60
SUCCESS_TMT = 600
FAIL_TMT = 60
ERR_TMT = 10
INET_HOST = '8.8.8.8'
```

Run python script
   ```sh
   python3 gate/gate.py
   ```
